# Testing Code Blocks

This is a test file to verify that code blocks are rendering properly in MDX files.

## JavaScript Example

```javascript
// This is a JavaScript code block
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet("World"));
```

## TypeScript Example

```typescript
// This is a TypeScript code block
function greet(name: string): string {
    return `Hello, ${name}!`;
}

console.log(greet("World"));
```

## Svelte Example

```svelte
<script>
    let name = "World";

    function greet(name) {
        return `Hello, ${name}!`;
    }
</script>

<h1>{greet(name)}</h1>
```

/* This is a CSS code block */
